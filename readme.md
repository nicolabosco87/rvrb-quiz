# RVRB Quiz Bot

A RVRB bot that asks trivia questions every track played.

| Command | Description |
| -quiz start | Starts the Quiz! |
| -quiz score [dateFilter] | prints out the current round score. DateFilter could be "month", "all" |
| -quiz stop | Ends the quiz|
